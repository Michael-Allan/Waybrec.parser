Waybrec Parser

This project implements a pull parser for Waybrec. It includes both a cursor based reader for efficient reading of Waybreccian markup, and an XML translator. The latter produces an XML homologue (XML-form Waybrec) that makes Waybrec accessible to the full suite of JAXP, the Java API for XML Processing.
