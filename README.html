Waybrec Parser

This project implements a cursor-based pull parser for efficient reading of Waybreccian markup. It includes both a basic Waybrec cursor that emits native parse states, and a translational cursor that emits X-Waybrec, an XML homologue that makes Waybrec accessible to the full suite of JAXP, the Java API for XML Processing.
